Cooperative transport logistics use case
----------------------------------------

This use case represents a robotic logistics production scenario, in which a plant operator will request efficient transports performed by robots.

    The use case is detailed in :octicon:`file-symlink-file;1em` :ref:`Cooperative transport logistics <Use case transport logistics>`.


Flexible small-scale manufacturing use case
-------------------------------------------


This use case represents a flexible small-scale manufacturing scenario, in which SMIA is leveraged to represent both reactive assets that offer the capabilities to meet manufacturing demands, and logical assets responsible for orchestration through the SMIA PE proposal. Furthermore, humans are integrated as part of the execution of manufacturing tasks, assuming a productive role rather than a coordinating one. Consequently, this scenario evaluates the automation of flexible production processes through a fully collaborative SMIA-based system.

    The use case is detailed in :octicon:`file-symlink-file;1em` :ref:`Flexible small-scale manufacturing <Use case flexible small-scale manufacturing>`.
